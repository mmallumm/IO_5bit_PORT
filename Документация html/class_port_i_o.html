Подробное описание

Класс для работы с 5-битным дискретным портом ввода-вывода микроконтроллера

Класс PortIO содержит: 10 методов; 5 констант (5 битовых флагов); 2 поля для работы с регистрами микроконтроллера (PORT - регистр питания линий порта, DDR - регистр направлений линий порта).

Конструктор класса

Конструктор принимает два аргумента-указателя "*port" и "*ddr", присваивает их значения указателям PORT и DDR соответственно.

Конструктор присваивает регистрам PORT и DDR значение по умолчанию (0b00000).

◆ READ_PIN_DIRECTION() [1/2]

bool PortIO::READ_PIN_DIRECTION ( const unsigned char  option)

Метод чтения направления линии порта

Для корректной работы входной параметр "option" должен представлять из себя 5 битовый флаг PINx, где x - индекс нужной линии (от 0 до 4).

Метод возвращает 1 в случае установленного направления линии "Выход", возвращает 0 в случае установленного направления линии "Вход".
